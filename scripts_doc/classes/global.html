saveFile

(
  • content
  • [filename]
)
String

Save content to the file. If no file name will be specified, then promp will be opened and ask user for file name.

Parameters:

  • content String

    What to save to file

  • [filename] String optional

    path to file

Returns:

String: Path to file name

Example:

var path = saveFile('this string will be in file');
var fileContents = openFile(path); # no prompt here
console.write(fileContents) # we should see "this string ..."
